Projeto Cloud Azure

Descrição

Este projeto foi desenvolvido com o objetivo de aplicar na prática conceitos de Cloud Computing utilizando Microsoft Azure. Foram provisionadas duas máquinas virtuais (Windows Server e Ubuntu Linux), configurando comunicação entre elas, acesso remoto e serviços web. Durante o projeto enfrentei desafios na criação de máquinas virtuais, configuração de acesso remoto (RDP/SSH) e controle de portas via firewall (NSG). Isso me permitiu aplicar na prática conceitos de redes e segurança, além de desenvolver habilidades de resolução de problemas em ambiente real de cloud.

Arquitetura

VM Windows Server: 10.0.0.4

VM Ubuntu: 10.0.0.5

Rede: VNet Azure

Funcionalidades

  • Acesso remoto via RDP (Windows)
  • Acesso via SSH (Linux)
  • Comunicação interna entre VMs
  • Servidor web rodando (IIS e Python)
  • Firewall configurado (NSG)
  • Teste de portas com Nmap

Segurança (básico)

Foram realizados testes de abertura e fechamento de portas, além de análise de serviços expostos utilizando Nmap, simulando cenários básicos de segurança.
